## 🚀 Instalando Ecommerce

Para instalar o Ecommerce, siga estas etapas:

```
npm install --legacy-peer-deps
```

## ☕ Usando Ecommerce

Para usar Ecommerce, siga estas etapas:

```
npm run dev
```
